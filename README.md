# Sales Data Analysis Project

A data analysis project using Python and SQL to extract business insights from sales data.

## Tools Used
- Python (Pandas, NumPy)
- SQL (SQLite)
- Matplotlib & Seaborn
- Jupyter Notebook

## Project Structure
- `data/` - Sales dataset and SQLite database
- `notebooks/` - Jupyter notebook with full analysis
- `outputs/` - Generated charts and visualizations
- `sql/` - SQL queries

## Key Insights
- Analysed 500 sales records across 5 product categories
- Total Revenue: $1,256,643
- Top Product Category: Books
- Top Region: North

## How to Run
```bash
pip install -r requirements.txt
jupyter notebook notebooks/analysis.ipynb
```
